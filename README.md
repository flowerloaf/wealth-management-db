# Wealth Management DB

## Setup
1. Install PostgreSQL
2. Create database: `wealth_management`
3. Run:
   - `database/schema.sql`
   - `database/seed_test_data.sql`
   - `database/sample_queries.sql`

## Project structure
- `database/` SQL setup files
- `data/` CSV data
- `scripts/` generation scripts
- `cli/` Java command line interface
- `app/` Spring Boot web app
